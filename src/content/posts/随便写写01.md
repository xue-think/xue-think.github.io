### 毕业

首先，我终于 public defense 了，毕业了。翻看 defense 那几天（12/2）写的东西：

Defense Day :

Defense 过了，我觉得自己 defense 表现非常好。笑话开头，笑话结束。带的饮料、点的三个 pizza 现场的人刚好吃光，还剩一点零食，正好留给 lab.

结束之后吃饭，然后回去补眠 3小时，毕竟 defense 前一晚还是有点紧张的，而且其实我前几天嗓子就很干了。但是我的肾上腺素（感谢TA）硬是让我 defense 的时候精神满满。

我本来以为我会很开心的（当然也开心，但是并非是我预想的那种在家大喊大叫，两岸猿声啼不住的开心，在家边发出猴子一样的笑声边喊： 多比 is free！因为我自由了，自由了，自由了！终于可以想不做什么就不做什么了）。结果呢？生病了，病来如山倒，而且发现还有需要签字的表格，老板还让我做一下 code release...

Day 2~3:

病来如山倒真不是吹，Defense 结束的晚上开始有点偏头痛的症状，马上吃了药，开始睡觉，睡衣T恤的领口被汗湿了好几轮，才到了早上。撑着做了点行政上的事（因为有 deadline），玩了会游戏（轻伤不下火线我懂了，长辈腰酸背痛也要去打麻将我懂了），然后又开始睡，直到睡饱有点感伤，因为得要开始准备搬家的事情了，P正好要回国休假一段时间，感伤+2。又有点像大学期末考结束之后，室友已经买好了回家乡的车票，她得先走了，但我的火车还有几天，又感伤又自由的感觉，寝室里/楼道里的人逐渐减少，那时候，我会买一大包「呀！土豆」和泡面🍜，然后在宿舍狂看「康熙来了」或者美剧，「Gossip Girl」的第一季就是那个时候一天一夜看完的，我可真厉害୧(๑•̀◡•́๑)૭

但生病又让我感觉像中考结束之后，中考结束之后也是生了场病，在床上躺了两天，感觉整个骨头都被烧软了，同样印象深刻的也是--睡衣会被汗湿几轮，那个时候，大概也是身体一直撑着，直到中考结束，才告诉我：你可以放松一下了，放下了。那个时候是我搬去和我妈住了大概两年吧，我妈：在我成绩不好的时候，会言语贬低，在我成绩好的时候，会有表面的和平（但依旧是 emotional unavailable 的）。所以我被迫变成了成绩好的人，当然后面我也是有体会到一些学习の乐趣的，但 conditionally love 和 conditionally acceptance 这我熟啊，「有用才能配得上爱」，这我太熟了。

感伤的时候觉得人是被困在以前的某些时刻了，越活越大 = 越活越小，曾经的，童年的刺痛不肯放过我。但感伤过去的时候，会觉得啊，这是给我重来的机会嘛，那个时候我没有被我妈接受、支持，现在有P接受、支持我，而且最重要的，我对于我自己，我不需要有用才能配得上爱，或者「有用」到底是什么意思呢？社会定义的「有用」吗？好，我要堂堂正正的说：我觉得我对于社会，做的已经够多了，我完全不欠社会任何东西了，我只需要做我觉得「有趣」的事情就可以，做我自己想做的即可。（哈哈哈哈，可真狂啊，虽然话是这样说，但我知道，以后总会有小小的妥协和选择的时候，而且可能做自己想做的事也需要更多努力？一些不自由才能带来自由，所谓 self-displicline 才能带来解放和自由）

Anyway，这些东西把我从感伤中拉了出来，以及：此时病还没有好透的我依旧缺觉，我还需要睡觉，睡觉，睡觉...zzz


### 论文

毕业前的最后一个项目有点遗憾，就是没中投的 conference，12月中出结果，没中。我一看 not accept 就有点失望，直接关掉电脑，不过我还是和老板表示：近期有关这个项目的小忙我可以帮（潜台词：做实验写 code 别叫我，就是改改文字什么的我 ok）结果老板说：你可以看一看 ac 意见，虽然这个没中，但 ac 表示，你来投咱们的 journal 吧，我们就直接算你是 major revision 了，然后你做到 rebuttal 中承诺要做的事的话，我们会尽量保留 reviewer continuity，这，我是没想到，那还可以啊，那就约好年后和老板和 collaborator 再开个会，我的预期是可能是需要花个两三周来 revise paper，至于 code，我其实是想再整理一下，那个 repository 有很多没用的代码，因为这个项目“转型”了好几次，😒，所以里面真的好些没用的代码，感觉可以改一下，而且代码因为有 Claude 、 ChatGPT 的帮忙，简直了，注释一会中文、一会英文，anyway，正好我的确还想再整理一下，不然 release code 我感觉会稍微有点羞耻（实际上：谁看啊，你这影响力没那么大，不过老实说，我蛮喜欢这个项目的 idea，虽然最后做的有点一般，希望哪个大牛看到这个 idea 和 paper 用其出色的能力来解决一些我没解决的问题，把我的 idea 发扬光大，我真的很喜欢这个 idea，但是就是能力有限，没有做成更 genealize 的解法）


### 项目

我开始做自己的项目了（完全属于自己，因为自己想做才做的），✌🏻，不知道最终会做成怎么样，但 let it be 吧，想做就做，感觉可以写 devlog，o(╯□╰)o，可以写在这里，看了一眼第一个 commit 是 12/5，但应该比这稍微早点，大概是 12/5 变成了 git repo。我还是对进度满满意的😊，之前有同学问到我在做什么，忍不住导出了一个 video 给她看（12/9），之后又给另一个同学看了，今天发现和上次的导出又有蛮多的不同了。因为目前所有的素材都是 AI 生成（thank you，ChatGPT，Copilot，Gemini，反正是三个都在用，谁生成的我满意就用谁的），让我震惊的是居然 Copilot 生成的素材最合我心意。但问题就是 AI 很难连贯和一致，即使 Google 的 Nano Banana 也没有做到，我对它期待还是蛮高的，因为之前看到别人说跑分很高什么的。
